# Tic-Tac-Toe (Java Swing)

A polished, dark-themed Tic-Tac-Toe game built with Java Swing.

## Features
- Clean dark UI with hover effects
- Live scoreboard (X wins / O wins / Ties)
- Winning line highlighted in green
- Tie game highlighted in yellow
- One-click Restart button (no need to relaunch)

## How to Run
```bash
javac App.java TicTacToe.java
java App
```

## Tech
- Java (Swing, AWT)
- No external dependencies
